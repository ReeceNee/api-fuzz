
# apiFuzz

Python JSON API Fuzzer based on PyJFuzz, PyJFAPI and uncurl

original:  [https://github.com/smasterfree/api-fuzz](https://github.com/smasterfree/api-fuzz)

## Installation

```
pip install PyJFuzz

git clone git@github.com:ReeceNee/api-fuzz.git
```
